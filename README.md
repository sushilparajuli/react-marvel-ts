<br>

## 🚀 How to run project ?

Clone the project.

```bash
$ git clone
$ cd
```

To start it, follow the steps below::
```bash
# install the dependencies
$ yarn / npm install

# Run the project in development
$ yarn dev / npm run dev
```

To build the project::
```bash
$ yarn build / npm run build
```%
